In this experiment you will hear a number of sounds.

If the sound is a word, press the F key.

If the sound is not a word, press the J key.

Please use your left index finger to push the F key, and your right index finger to push the J key.

You will need to complete the experiment in one sitting, which will take between 15-20 minutes.

There is one break during the experiment.

Please do not start the experiment if you do not have enough time to complete it.

We will start with some practice.

If you are having trouble loading the files, please try using a Chrome or Firefox browser.
